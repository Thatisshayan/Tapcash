# TapCash — Model U Final Master Base

This package contains the final selected TapCash landing-page direction and a refined wallet-tap logo system.

## What this includes

- Vite + React landing page
- Framer Motion dependency for motion-ready UI
- Final wallet-tap logo SVGs
- Alternative logo SVG concepts
- Design spec and dev handoff
- Organized source files for developers

## Run locally

```bash
npm install
npm run dev
```

Open the local Vite URL in your browser.

## Build

```bash
npm run build
npm run preview
```

## Current final direction

**Model U — Calm Premium Conversion**

Hero copy:

> Play. Earn. Cash Out.

Core mechanics:

- CashPath™ — shows reward progress from offer to payout
- TapScore™ — shows safest offers before user starts
- Start My Safest Offer — first action for new users
- Three featured offers only above the fold

## Logo direction

Final logo direction is **Wallet Tap**:

- Wallet = cashout / payout
- Tap ripple = easy action / tap to start
- Cash edge = real rewards
- No full hand cursor in final icon to keep it premium and scalable
