# TapCash Model U Design Spec

## Brand personality

Friendly fintech gaming. Calm, premium, reward-focused. Avoid casino, sportsbook, crypto-bro, and aggressive esports energy.

## Final page structure

1. Header
2. Hero: Play. Earn. Cash Out.
3. CTA: Start My Safest Offer
4. Three Top Offers
5. CashPath™ Live
6. See It In Action
7. TapScore™
8. Minimal trust strip

## Color system

- Background: `#050813`
- Panel: `rgba(9,16,31,.74)`
- Cyan: `#18D9FF`
- Purple: `#7C3DFF`
- Green: `#31F06F`
- Yellow: `#FFC442`
- Text: `#F6F8FF`
- Muted text: `#9AA8C6`

## Copy rules

Use less text. Keep everything concrete.

Do:
- Complete verified offers.
- Track every step.
- Cash out when rewards clear.
- Know the safest offers before you start.

Avoid:
- Real people. Real payouts.
- Millions earning daily.
- Overly fake social proof.
- Unverified numbers.

## Motion notes

Use subtle Framer Motion:
- Hero fade + rise on load
- CTA hover lift
- Offer cards hover y: -6px
- CashPath steps staggered reveal
- TapScore ring subtle pulse once on entry

Do not over-animate. This is premium, not arcade chaos.
