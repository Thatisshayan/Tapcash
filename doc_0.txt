You just need a page with enough space to run it on your webpage/webapp. You’ll need to dynamically generate the Rewards Center link based on the RapidoReach UID. This ensures that every user gets the right credit for their activity. First generate the RapidoReach UID using a script on the server, and pass that to the website. Important: Build UID as internalUserId-appId-checksum where checksum = md5(&quot;internalUserId-appId-appKey&quot;) . URL-encode the final UID before placing it in the userId query parameter.
